Tentang Sakuwise

Sakuwise adalah aplikasi anggaran pribadi untuk Android yang dirancang dengan prinsip local-first. Semua data keuangan Anda — transaksi, akun, plan, foto struk, info aset — tinggal di perangkat Anda. Tanpa server. Tanpa analytics. Tanpa pelacakan.

🔒 Encrypted at Rest

Database terenkripsi dengan AES-256 via SQLCipher. Kunci tersimpan di Android Keystore.

💾 Backup yang Anda Kontrol

Simpan backup ke lokasi pilihan Anda atau opt-in ke Google Drive private folder.

📷 OCR Struk On-Device

ML Kit Text Recognition memproses struk Anda di perangkat — tanpa upload ke server.

🌱 Pixel-Perfect Design

Forest green warm Indonesia-first. Dibuat untuk pengguna Indonesia kelas menengah.
